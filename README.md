A simple, terminal-based implementation of the classic Snake game.

**How to Play:**

1.  Navigate to the `snake-cli` directory:
    ```bash
    cd snake-cli
    ```
2.  Run the game:
    ```bash
    python3 snake.py
    ```

**Controls:**

-   **Up:** `z` or `Up Arrow`
-   **Down:** `s` or `Down Arrow`
-   **Left:** `q` or `Left Arrow`
-   **Right:** `d` or `Right Arrow`
